우선 프로그램을 실행하기 위해서 윈도우 방화벽을 꺼주셔야 합니다. 코드만 실행시 윈도우 방화벽을 끄지 않아도 됩니다. 

실행 방법
1. 파일선택 버튼을 눌러 파일(들)을 선택합니다.
2. 기법선택에서 원하는 암호화/복호화 기법을 선택합니다
3. 암호를 직접정하고 싶다면 암호를 입력합니다.
4. 원본파일을 삭제하고 싶을시 체크를 해줍니다.
5. 예약시간을 정하고 싶다면 양식에 맞게 예약시간을 작성해줍니다. 원치 않을시 미입력합니다.
6.실행버튼을 누르면 실행이 됩니다. 예약시간을 작성했을경우 실행버튼을 눌러둘시 그시간에 작동을 합니다.
